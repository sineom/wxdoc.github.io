取二维码

http://127.0.0.1:8898/v1/Login/GetQRcode?isIPad=0 //获取Windows微信二维码 使用PC协议 封号尚可 小概率
http://127.0.0.1:8898/v1/Login/GetQRcode?isIPad=1 //获取iPad微信二维码 使用iPad协议 暂不可用 封号严重
http://127.0.0.1:8898/v1/Login/GetQRcode?isIPad=2 //获取Mac微信二维码 使用Mac协议 封号尚可 小概率
http://127.0.0.1:8898/v1/Login/GetQRcode?isIPad=2&ip=127.0.0.1:8888&user=&pass=&wxid=wxid_xxxx //Socks5 登陆
http://127.0.0.1:8898/v1/Login/GetQRcode?isIPad=2&wxid=wxid_xxxx //载入wxid设备信息获取二维码
如果不传usesync=1参数程序将关闭每个账号的消息同步 SNS同步 插件解析等可以节省一CPU资源 不适用消息类、 机器人等业务！
这样打开同步功能
http://127.0.0.1:8898/v1/Login/GetQRcode?isIPad=2&usesync=1
或者
账号登录成功后设置
http://127.0.0.1:8898/v1/LuaApiCaller?funcname=MagicCgi&timeout=10&wxid=wxid_12356789
POST JSON
{ "CgiCmd":1000000005, "CgiRequest":{ "UseSync":false } }
true 打开同步功能 false 关闭同步功能
后续所有API接口的wxid=wxid_xxxxx的参数均为机器人Wxid
登陆成功后 掉线必须执行历时登陆接口以防封号
或者通过载入Wxid设备信息获取二维码进行扫码登陆 这样才可以保持一号一设备
